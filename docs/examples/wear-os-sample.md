# Wear OS Sample

The Wear OS sample shows a standalone Wear OS app using SalusChart's watch-optimized chart components.

Repository: [SalusChart-WearOS-Sample](https://github.com/HDIL-YS/SalusChart-WearOS-Sample)

Use the sample to inspect:

- small-screen chart layouts
- circular viewport composition
- minimal chart variants
- watch-friendly health summaries

For chart APIs, see [Wear OS Charts](../charts/wear-os-charts) and [Wear OS Examples](./wear-os).
