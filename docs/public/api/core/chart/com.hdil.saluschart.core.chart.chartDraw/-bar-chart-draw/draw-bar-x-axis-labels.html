chart/com.hdil.saluschart.core.chart.chartDraw/BarChartDraw/drawBarXAxisLabels

drawBarXAxisLabels

fun drawBarXAxisLabels(ctx: DrawContext, labels: List<String>, metrics: ChartMath.ChartMetrics, centered: Boolean = true, textSize: Float = 28.0f, maxXTicksLimit: Int? = null, xLabelAutoSkip: Boolean = false, labelYOffset: Float = 50.0f)

Draws X-axis tick labels for a bar chart.

Notes:

  • Labels are anchored using the bar-slot spacing, so the first label starts at paddingX + (barWidth / 2) and proceeds by slot spacing.

  • If xLabelAutoSkip is enabled, labels may be skipped to avoid overlapping.

Parameters

ctx

Draw context whose native canvas is used to render text.

labels

Full list of labels (one per bar slot).

metrics

Chart layout metrics used for positioning.

centered

Whether text is center-aligned (default: true).

textSize

Text size in pixels (default: 28f).

maxXTicksLimit

Optional cap on how many labels can be displayed (null = no cap).

xLabelAutoSkip

If true, compute an auto-skip factor based on text width and chart width.
