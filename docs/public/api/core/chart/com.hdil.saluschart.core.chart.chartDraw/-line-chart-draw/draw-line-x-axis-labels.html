chart/com.hdil.saluschart.core.chart.chartDraw/LineChartDraw/drawLineXAxisLabels

drawLineXAxisLabels

fun drawLineXAxisLabels(ctx: DrawContext, labels: List<String>, metrics: ChartMath.ChartMetrics, centered: Boolean = true, textSize: Float = 28.0f, maxXTicksLimit: Int? = null, xLabelAutoSkip: Boolean = false)

Draws X-axis labels for a line chart.

Positioning rule:

  • Labels are centered within each "slot" using (index + 0.5) * spacing, where spacing = chartWidth / totalLabelCount.

Label density:

  • If xLabelAutoSkip is enabled, labels may be skipped using ChartMath.computeAutoSkipLabels to reduce overlap.

Clamping behavior (when centered is true):

  • The label's center X is clamped so the text does not overflow the canvas bounds.

Note:

  • The current implementation uses the line-chart spacing rule and places the first label half a slot away from the left edge. This matches existing point positioning behavior.

Parameters

ctx

Draw context whose native canvas is used to render text.

labels

Full list of labels (one per x-position).

metrics

Chart metrics used for padding and chart dimensions.

centered

If true, center-align the text (default: true).

textSize

Text size in pixels (default: 28f).

maxXTicksLimit

Optional cap on how many labels can be displayed (null = no cap).

xLabelAutoSkip

If true, compute an auto-skip factor based on text width and chart width.
