drawPieSection

fun drawPieSection(drawScope: DrawScope, center: Offset, radius: Float, startAngle: Float, sweepAngle: Float, color: Color, isDonut: Boolean, strokeWidth: Float, isSelected: Boolean = false, animationScale: Float = 1.0f, alpha: Float = 1.0f)

Draws a single pie (or donut) section.

Behavior:

  • When isSelected is true, the section is slightly offset outward from the center and can be scaled using animationScale.

  • When isDonut is true, the section is drawn as a stroked arc (ring segment).

  • When isDonut is false, the section is drawn as a filled wedge and includes two white divider lines at its boundaries.

Parameters

drawScope

Compose draw scope.

center

Center of the pie in canvas coordinates.

radius

Base radius of the pie in pixels.

startAngle

Start angle in degrees.

sweepAngle

Sweep angle in degrees.

color

Section color.

isDonut

If true, draw as a donut (ring segment) using strokeWidth.

strokeWidth

Stroke width (pixels) used only when isDonut is true.

isSelected

If true, offset the section outward from the center.

animationScale

Scale factor applied to radius (1.0f = no scaling).

alpha

Alpha applied to color (1.0f = fully opaque).
