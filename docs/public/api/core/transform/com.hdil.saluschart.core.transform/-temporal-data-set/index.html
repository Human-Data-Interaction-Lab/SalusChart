TemporalDataSet

data class TemporalDataSet(val x: List<Instant>, val y: List<Double>? = null, val yMultiple: Map<String, List<Double>>? = null, val timeUnit: TimeUnitGroup = TimeUnitGroup.HOUR)

Normalized time-series container used internally by SalusChart.

This class represents time-indexed numeric data before it is mapped to chart-specific models (e.g., ChartMark).

Invariants:

  • Either y (single-value series) OR yMultiple (multi-value series) must be provided.

  • Not both.

  • All value lists must match the size of x.

Parameters

x

Time axis values (Instant).

y

Single-value series (nullable if multi-value).

yMultiple

Multi-value series map (nullable if single-value).

timeUnit

Granularity of the dataset.
