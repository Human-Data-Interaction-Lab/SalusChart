BarChart

@Composable
fun BarChart(modifier: Modifier = Modifier, data: List<ChartMark>, xLabel: String = "Time", yLabel: String = "Value", title: String = "Bar Chart Example", barColor: Color = Color.Unspecified, minY: Double? = null, maxY: Double? = null, barWidthRatio: Float = 0.8f, xLabelTextSize: Float = 28.0f, tooltipTextSize: Float = 32.0f, yAxisPosition: YAxisPosition = YAxisPosition.LEFT, interactionType: InteractionType.Bar = InteractionType.Bar.BAR, onBarClick: (Int, Double) -> Unit? = null, showTitle: Boolean = false, showYAxis: Boolean = true, showLabel: Boolean = false, xLabelAutoSkip: Boolean = true, maxXTicksLimit: Int? = null, yTickStep: Double? = null, unit: String = "", referenceLines: List<ReferenceLineSpec> = emptyList(), showYAxisHighlight: Boolean = false, windowSize: Int? = null, contentPadding: PaddingValues = PaddingValues(16.dp), pageSize: Int? = null, unifyYAxisAcrossPages: Boolean = true, initialPageIndex: Int? = null, yAxisFixedWidth: Dp = 20.dp, barCornerRadiusFraction: Float = 0.0f, barCornerRadiusFractions: BarCornerRadiusFractions? = null, roundTopOnly: Boolean = false, showLegend: Boolean = false, legendPosition: LegendPosition = LegendPosition.BOTTOM, legendLabel: String = "", tooltipColor: Color = Color.Unspecified)

Displays a vertical bar chart for the provided data, with support for interactive tooltips, optional horizontal scrolling, paged navigation, a configurable Y-axis, reference lines, and a legend.

When pageSize is set and the data exceeds that value, the chart automatically switches to a horizontally pageable layout with a shared Y-axis across all pages.

Parameters

modifier

Modifier applied to the outermost layout container.

data

List of ChartMark entries to render as bars. Returns early if empty.

xLabel

Text label displayed below the X-axis.

yLabel

Text label displayed alongside the Y-axis.

title

Chart title shown above the chart area when showTitle is true.

barColor

Fill color applied to every bar.

minY

Optional lower bound for the Y-axis; computed from data when null.

maxY

Optional upper bound for the Y-axis; computed from data when null.

barWidthRatio

Fraction of each slot's width occupied by the bar (0.0–1.0).

xLabelTextSize

Text size in pixels for X-axis tick labels.

tooltipTextSize

Text size in pixels for the tooltip label.

yAxisPosition

Side on which the Y-axis is drawn (YAxisPosition.LEFT or YAxisPosition.RIGHT).

interactionType

Controls bar tap behaviour: InteractionType.Bar.BAR makes the bar itself tappable; InteractionType.Bar.TOUCH_AREA uses an invisible full-height tap region.

onBarClick

Optional callback invoked with the tapped bar's index and Y value.

showTitle

Whether to render the title above the chart.

showYAxis

Whether to draw the Y-axis line and tick labels.

showLabel

Whether to display the data value label on top of each bar.

xLabelAutoSkip

When true, overlapping X-axis tick labels are automatically skipped.

maxXTicksLimit

Optional cap on the number of X-axis tick labels rendered.

yTickStep

Fixed interval between Y-axis grid lines; auto-calculated when null.

unit

Unit string appended to tooltip values (e.g. "kg", "bpm").

referenceLines

Horizontal reference lines drawn across the plot area.

showYAxisHighlight

When true, reference-line values are highlighted on the Y-axis.

windowSize

Number of data points visible at once in free-scroll mode; enables horizontal scrolling when smaller than the total data size. Mutually exclusive with pageSize.

contentPadding

Padding applied around the chart content area.

pageSize

Number of data points shown per page; enables the pager when data exceeds this value. Mutually exclusive with windowSize.

unifyYAxisAcrossPages

When true, all pages share the same Y-axis range in paged mode.

initialPageIndex

Page to display first in paged mode; defaults to the last page when null.

yAxisFixedWidth

Width reserved for the external Y-axis pane in scroll or highlight mode.

barCornerRadiusFraction

Uniform corner radius for all bar corners as a fraction of bar half-width.

barCornerRadiusFractions

Per-corner radius fractions; overrides barCornerRadiusFraction when provided.

roundTopOnly

When true, only the top corners of each bar are rounded.

showLegend

Whether to display a legend entry near the chart.

legendPosition

Where to place the legend relative to the chart.

legendLabel

Text shown in the legend for this data series.

tooltipColor

Background color of the tooltip bubble; defaults to barColor.
