CalendarChart

@Composable
fun CalendarChart(modifier: Modifier = Modifier, entries: List<CalendarEntry>, yearMonth: YearMonth = YearMonth.now(), markerType: CellMarkerType = CellMarkerType.BUBBLE, bubbleType: BubbleType = BubbleType.CIRCLE, maxBubbleSize: Float = 10.0f, minBubbleSize: Float = 6.0f, color: Color = Color.Unspecified)

Renders a single-month calendar chart with per-day value markers.

Parameters

modifier

Modifier applied to the chart container.

entries

List of CalendarEntry values to display.

yearMonth

The month to display; defaults to the current month.

markerType

Marker style for each day cell: CellMarkerType.BUBBLE or CellMarkerType.MINI_RINGS.

bubbleType

Shape of the bubble marker: BubbleType.CIRCLE or BubbleType.RECTANGLE.

maxBubbleSize

Maximum bubble radius in dp.

minBubbleSize

Minimum bubble radius in dp.

color

Default marker color; overridden per-entry by CalendarEntry.color when set.
