HorizontalStackedBarChartList

@Composable
fun HorizontalStackedBarChartList(modifier: Modifier = Modifier, title: String, datePeriodText: String? = null, rows: List<HorizontalStackedBarRow>, colors: List<Color> = emptyList(), barTrackColor: Color = Color(0xFFF1F1F1), onRowClick: (Int, Int?, Float) -> Unit? = null, showLegend: Boolean = false, legendPosition: LegendPosition = LegendPosition.BOTTOM, legendLabels: List<String> = emptyList())

Renders a vertical list of horizontal stacked bar rows with an optional legend and tap tooltip.

Each row visualises one HorizontalStackedBarRow as a pill-shaped track subdivided into colored segments. Tapping a segment shows a tooltip with the segment value; tapping elsewhere dismisses it.

Parameters

modifier

Modifier applied to the outer container.

title

Chart title displayed at the top.

datePeriodText

Optional date-range subtitle displayed below the title.

rows

Ordered list of row data to render.

colors

Colors assigned to segments by index; cycles if fewer than the maximum segment count.

barTrackColor

Background track color for unfilled portions of each bar.

onRowClick

Optional callback invoked when a segment is tapped. Receives (rowIndex, segmentIndex, segmentValue).

showLegend

Whether to display a legend.

legendPosition

Position of the legend relative to the chart content.

legendLabels

Labels for legend entries; must align with colors by index.
