MiniActivityRings

@Composable
fun MiniActivityRings(modifier: Modifier = Modifier, rings: List<ProgressChartMark>, colors: List<Color>, strokeWidth: Float, maxLaps: Int = 2, trackAlpha: Float = 0.2f, gapRatio: Float = 0.14f, startAngle: Float = -90f, shadowAlpha: Float = 0.1f, shadowExtraWidthRatio: Float = 0.45f)

Renders a compact set of concentric activity rings, similar to the Apple Watch Activity display.

Each ring corresponds to one ProgressChartMark and fills proportionally to current / max. Progress beyond 100 % wraps up to maxLaps full rotations.

Parameters

modifier

Modifier applied to the Canvas.

rings

Ordered list of progress marks; index 0 is the outermost ring.

colors

Colors for each ring by index; cycles if fewer than rings size.

strokeWidth

Ring stroke width in pixels.

maxLaps

Maximum number of full rotations allowed (e.g., 2 = up to 200 %).

trackAlpha

Alpha applied to the background track arc of each ring.

gapRatio

Gap between rings as a fraction of strokeWidth.

startAngle

Starting angle in degrees (−90 = 12 o'clock).

shadowAlpha

Alpha of the drop shadow drawn behind each filled arc.

shadowExtraWidthRatio

Extra width of the shadow stroke relative to strokeWidth.
