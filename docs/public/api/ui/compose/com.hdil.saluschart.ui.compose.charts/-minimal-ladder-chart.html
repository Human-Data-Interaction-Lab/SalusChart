MinimalLadderChart

@Composable
fun MinimalLadderChart(modifier: Modifier = Modifier, bandCount: Int = 3, selectedBandIndex: Int, markerRatio: Float, trackColor: Color, selectedColor: Color, markerColor: Color, bandHeight: Dp = 10.dp, bandGap: Dp = 2.dp, cornerRadiusRatio: Float = 0.95f, markerRadius: Dp = 4.dp, markerRingWidth: Dp = 2.dp, markerRingColor: Color = Color.White)

Renders a ladder (multi-band) chart where one band is highlighted and a positional marker dot is drawn within it.

Bands are stacked vertically and centered within the available height. The selected band is filled with selectedColor; all others use trackColor.

Parameters

modifier

Modifier applied to the Canvas.

bandCount

Total number of horizontal bands to draw.

selectedBandIndex

Index of the active band (0 = top).

markerRatio

Horizontal position of the marker dot within the selected band (0 = left, 1 = right).

trackColor

Color of inactive bands.

selectedColor

Color of the active band.

markerColor

Fill color of the marker dot.

bandHeight

Height of each band.

bandGap

Vertical gap between consecutive bands.

cornerRadiusRatio

Corner radius of each band as a fraction of bandHeight.

markerRadius

Radius of the marker dot.

markerRingWidth

Width of the white ring drawn around the marker dot.

markerRingColor

Color of the ring surrounding the marker dot.
