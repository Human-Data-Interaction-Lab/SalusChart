MinimalMultiSegmentGauge

@Composable
fun MinimalMultiSegmentGauge(modifier: Modifier = Modifier, segments: List<MinimalGaugeSegment>, markerRatio: Float, label: String = "높음", bubbleColor: Color = Color(0xFFD6F5DF), bubbleTextColor: Color = Color(0xFF0A7A32), barHeight: Dp = 14.dp, cornerRadius: Dp = 999.dp, markerWidth: Dp = 44.dp, markerHeight: Dp = 22.dp, markerColor: Color = Color(0xFF7BE64C), markerShadowAlpha: Float = 0.1f, bubblePaddingH: Dp = 10.dp, bubblePaddingV: Dp = 6.dp, bubbleCornerRadius: Dp = 18.dp, pointerWidth: Dp = 14.dp, pointerHeight: Dp = 8.dp, bubbleGapFromBar: Dp = 8.dp, bubbleTextSizeSp: Float = 12.0f)

Renders a compact horizontal multi-segment gauge bar with a floating capsule marker.

The bar is divided into proportional colored zones defined by segments. A rounded-rectangle marker is drawn at the horizontal position specified by markerRatio, constrained to remain within the segment it falls inside.

Parameters

modifier

Modifier applied to the Canvas.

segments

Ordered list of gauge segments that partition the bar from left to right.

markerRatio

Position of the marker along the gauge (0 = left edge, 1 = right edge).

label

Text displayed inside the callout bubble.

bubbleColor

Background color of the callout bubble.

bubbleTextColor

Text color inside the callout bubble.

barHeight

Height of the gauge bar.

cornerRadius

Corner radius of the gauge bar; defaults to fully rounded (pill).

markerWidth

Requested width of the marker capsule; clamped to the containing segment width.

markerHeight

Height of the marker capsule.

markerColor

Fill color of the marker capsule.

markerShadowAlpha

Alpha of the drop shadow rendered below the marker.

bubblePaddingH

Horizontal padding inside the bubble.

bubblePaddingV

Vertical padding inside the bubble.

bubbleCornerRadius

Corner radius of the bubble rectangle.

pointerWidth

Width of the triangular pointer.

pointerHeight

Height of the triangular pointer.

bubbleGapFromBar

Gap between the bottom of the bubble pointer and the top of the marker.

bubbleTextSizeSp

Font size of the bubble label in sp.
