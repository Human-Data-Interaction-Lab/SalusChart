MinimalSleepChart

@Composable
fun MinimalSleepChart(modifier: Modifier = Modifier, columns: List<SleepColumn>, barWidthRatio: Float = 0.55f, columnGapRatio: Float = 0.55f, segmentGapRatio: Float = 0.3f, trackColor: Color = Color(0xFFEFEFEF), trackAlpha: Float = 0.0f, cornerRadiusRatio: Float = 0.45f, maxValueOverride: Float? = null)

Renders a compact multi-column sleep chart where each column is a vertical stack of rounded-rectangle sleep stage capsules.

Columns are placed side-by-side with equal slot widths. Heights are normalized so the tallest column fills the available canvas height (or maxValueOverride if provided).

Parameters

modifier

Modifier applied to the Canvas.

columns

Ordered list of day columns to render.

barWidthRatio

Width of each capsule as a fraction of its slot width.

columnGapRatio

Horizontal gap between columns as a fraction of the capsule width.

segmentGapRatio

Vertical gap between segments within a column as a fraction of the capsule width.

trackColor

Optional background track color drawn behind each column when trackAlpha> 0.

trackAlpha

Alpha of the background track; set to 0 to hide it.

cornerRadiusRatio

Corner radius of each capsule as a fraction of the capsule width.

maxValueOverride

Optional total value that maps to the full canvas height; computed from the tallest column when null.
