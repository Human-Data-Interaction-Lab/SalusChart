ProgressChart

@Composable
fun ProgressChart(modifier: Modifier = Modifier, data: List<ProgressChartMark>, title: String = "Progress Chart", isDonut: Boolean = true, isPercentage: Boolean = true, colors: List<Color>? = null, donutHeight: Dp = 200.dp, strokeWidth: Dp = 20.dp, barHeight: Dp = 16.dp, barSpacing: Dp = 10.dp, topPadding: Dp = 8.dp, bottomPadding: Dp = 8.dp, showLabels: Boolean = true, showValues: Boolean = true, showCenterInfo: Boolean = true, centerTitle: String = "Activity", centerSubtitle: String = "Progress", showLegend: Boolean = true, legendTitle: String? = null, legendColorBoxSize: Dp = 10.dp, legendTextSize: TextUnit = 12.sp, legendSpacing: Dp = 8.dp, legendPadding: Dp = 8.dp, interactionsEnabled: Boolean = true, tooltipEnabled: Boolean = true, tooltipFormatter: (ProgressChartMark) -> String? = null, tooltipBackgroundColor: Color = MaterialTheme.colorScheme.surface, tooltipTextColor: Color = MaterialTheme.colorScheme.onSurface, tooltipContent: @Composable (mark: ProgressChartMark, color: Color, modifier: Modifier) -> Unit? = null)

Displays a progress chart in either:

  • Donut (ring) mode

  • Horizontal bar mode

This composable supports multiple progress items. In donut mode, each item is rendered as a concentric ring. In bar mode, each item is rendered as a horizontal progress bar stacked vertically.

Parameters

modifier

Modifier applied to the chart container.

data

List of ProgressChartMark representing progress items. Each item should provide current value, maximum value, label, and optional unit.

title

Title text displayed above the chart.

isDonut

If true, renders a donut (ring) progress chart. If false, renders a horizontal bar progress chart.

isPercentage

If true, values are displayed as percentages. If false, raw values (current / max) are displayed.

colors

Optional list of colors used for progress items. If null, default colors will be generated automatically. If fewer colors than data items are provided, colors will repeat cyclically.

donutHeight

Height of the donut chart container. Only applied when isDonut is true.

strokeWidth

Thickness of each progress ring in donut mode. Ignored in bar mode.

barHeight

Height of each individual bar in bar mode. Ignored in donut mode.

barSpacing

Vertical spacing between bars in bar mode. Ignored in donut mode.

topPadding

Top padding inside the drawing area (bar mode only).

bottomPadding

Bottom padding inside the drawing area (bar mode only).

showLabels

If true, displays labels for each progress item.

showValues

If true, displays numeric progress values.

showCenterInfo

If true, displays center title and subtitle inside the donut chart. Ignored in bar mode.

centerTitle

Title text displayed at the center of the donut chart. Only used when isDonut is true and showCenterInfo is true.

centerSubtitle

Subtitle text displayed below centerTitle inside the donut chart.

showLegend

If true, displays a legend below the chart.

legendTitle

Optional title displayed above the legend.

legendColorBoxSize

Size of the color indicator box in the legend.

legendTextSize

Text size of legend labels.

legendSpacing

Horizontal spacing between legend items.

legendPadding

Vertical padding applied around the legend.

interactionsEnabled

Controls whether the chart responds to user gestures. When false, tap detection and selection behavior are disabled, and tooltips will not appear even if tooltipEnabled is true.

tooltipEnabled

Whether to show a tooltip on donut ring taps.

tooltipFormatter

Optional formatter for tooltip text. If provided, it overrides the default text.

tooltipBackgroundColor

Background color of the default tooltip. Ignored if tooltipContent is provided.

tooltipTextColor

Text color of the default tooltip. Ignored if tooltipContent is provided.

tooltipContent

Optional full tooltip composable override. If provided, it takes priority over tooltipFormatter and the default tooltip UI.
