RangeGaugeChart

@Composable
fun RangeGaugeChart(modifier: Modifier = Modifier, cornerRadius: Dp = 22.dp, minValue: Float, maxValue: Float, rangeStart: Float, rangeEnd: Float, recentValue: Float, unit: String = "bpm", recentLabel: String = "최근기록 오후 3:40", trackColor: Color = Color(0xFFF2F2F2), rangeColor: Color = Color(0xFFFF8A3D), textGray: Color = Color(0xFF9B9B9B), barHeight: Dp = 28.dp, titleFontSize: Int = 20, bottomFontSize: Int = 14, recentValueFontSize: Int = 16, subtitleFontSize: Int = 12, markerDotCount: Int = 7)

Renders a card-style range gauge that visualises a highlighted range on a linear scale and marks the most recent value with a dotted indicator.

The gauge bar spans minValue to maxValue. The segment between rangeStart and rangeEnd is filled with rangeColor. A column of white dots at recentValue's position indicates the current reading.

Parameters

modifier

Modifier applied to the Surface card.

cornerRadius

Corner radius of the card.

minValue

Minimum value of the gauge scale.

maxValue

Maximum value of the gauge scale.

rangeStart

Start of the highlighted range; clamped to minValue..maxValue.

rangeEnd

End of the highlighted range; clamped to minValue..maxValue.

recentValue

Most recent reading to mark on the gauge; clamped to minValue..maxValue.

unit

Unit string appended to values (e.g., "bpm").

recentLabel

Subtitle text shown below the gauge (e.g., "Latest recording 3:40 PM").

trackColor

Background track color for the gauge bar.

rangeColor

Fill color of the highlighted range segment.

textGray

Color used for the min/max boundary labels.

barHeight

Height of the gauge bar.

titleFontSize

Font size in sp for the range title (e.g., "60–100 bpm").

bottomFontSize

Font size in sp for the min/max boundary labels.

recentValueFontSize

Font size in sp for the recent-value label.

subtitleFontSize

Font size in sp for recentLabel.

markerDotCount

Number of white dots drawn in the marker column.
