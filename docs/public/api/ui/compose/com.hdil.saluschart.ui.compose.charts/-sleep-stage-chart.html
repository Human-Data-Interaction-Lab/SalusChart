SleepStageChart

@Composable
fun SleepStageChart(modifier: Modifier = Modifier, sleepSession: SleepSession, title: String = "Sleep Stage Analysis", showLabels: Boolean = true, showXAxis: Boolean = true, onStageClick: (Int, String) -> Unit? = null, barHeightRatio: Float = 0.5f, yAxisPosition: YAxisPosition = YAxisPosition.LEFT, contentPadding: PaddingValues = PaddingValues(horizontal = 24.dp, vertical = 16.dp), showTitle: Boolean = true, showYAxis: Boolean = true, showStartEndLabels: Boolean = true, xLabelAutoSkip: Boolean = true, yAxisFixedWidth: Dp = 48.dp)

Renders an Apple Health-style sleep stage chart displaying a SleepSession as colored horizontal capsules on a grid of four stage rows (Deep, Core, REM, Awake).

Consecutive stages of the same type are merged before rendering. Gradient connectors bridge transitions between adjacent capsules. Tapping a capsule shows a tooltip; tapping the same capsule again dismisses it.

Parameters

modifier

Modifier applied to the chart container.

sleepSession

Sleep session data containing the ordered list of SleepStage entries.

title

Chart title displayed when showTitle is true.

showLabels

Unused reserved parameter for future label control.

showXAxis

Whether to display the x-axis time labels below the chart.

onStageClick

Optional callback invoked when a capsule is tapped; receives (stageIndex, stageLabel).

barHeightRatio

Height of each capsule as a fraction of the row height.

yAxisPosition

Side on which the stage-name labels are drawn.

contentPadding

Padding applied around the entire chart.

showTitle

Whether to display title.

showYAxis

Whether to display the stage-name labels on the y-axis.

showStartEndLabels

Unused reserved parameter for start/end time label control.

xLabelAutoSkip

Unused reserved parameter for x-axis label skip logic.

yAxisFixedWidth

Width reserved for the y-axis label column.
